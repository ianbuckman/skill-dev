# ClipVault

简洁优美的 macOS Menu Bar 剪贴板管理工具，自动记录复制历史并支持快速检索。

## 安装

1. 双击 `ClipVault.dmg`
2. 将 `ClipVault.app` 拖入 `Applications` 文件夹
3. 右键点击 `ClipVault.app` → 打开（首次运行需要此步骤，因为未签名）

> **辅助功能授权**：如需使用全局快捷键（⌘+Shift+V），请前往 **系统设置 → 隐私与安全 → 辅助功能**，添加 ClipVault 并授权。

## 使用方法

### 基本操作
- **查看历史**：点击 Menu Bar 上的剪贴板图标，弹出历史面板
- **快捷唤起**：按 `⌘+Shift+V` 快速打开面板（需辅助功能授权）
- **复制条目**：在面板中点击任意条目，即可将其内容复制回系统剪贴板
- **搜索**：在搜索框中输入关键词，实时过滤文本条目

### 管理条目
- **固定条目**：悬停在条目上，点击 pin 图标将其固定。固定条目始终显示在列表顶部，不会被清空操作删除
- **取消固定**：悬停在已固定条目上，点击取消固定图标
- **删除条目**：悬停在条目上，点击垃圾桶图标删除该条目
- **清空历史**：点击底部 "Clear All" 按钮，清除所有未固定的条目

### 设置
点击标题栏齿轮图标打开设置窗口：
- **历史记录上限**：可选 100/200/500/1000 条（默认 500）
- **记录图片**：开关是否记录复制的图片内容
- **清空所有历史**：删除全部未固定记录

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| ⌘+Shift+V | 唤起/关闭剪贴板面板 |

## 数据存储

历史记录保存在 `~/Library/Application Support/ClipVault/history.json`。

用户偏好（历史上限、图片记录开关）保存在 UserDefaults 中。

## 卸载

1. 从 Applications 文件夹删除 ClipVault.app
2. 可选：删除 `~/Library/Application Support/ClipVault/` 目录清除历史数据

## 技术信息

- 平台：macOS 14+ (Sonoma)
- 技术栈：Swift 5.10 + SwiftUI
- 架构：Menu Bar App（LSUIElement）
- 构建：`swift build -c release`
