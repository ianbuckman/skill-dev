# App Concept: SnapCraft

## 一句话描述
一款功能全面的 macOS 截图录屏工具，媲美 CleanShot X，无需云服务，所有数据本地存储。

## 核心功能

### 截图功能
1. 区域截图 - 拖拽选择区域截图
2. 全屏截图 - 捕获整个屏幕
3. 窗口截图 - 点击选择窗口截图
4. 滚动截图 - 捕获可滚动内容（如长网页、聊天记录、代码）
5. 定时截图 - 支持延迟定时器（3秒、5秒、10秒）
6. 自拍叠加 - 截图时叠加摄像头画面

### 录屏功能
1. 全屏录制 - 录制整个屏幕
2. 区域录制 - 选择区域录制
3. GIF 录制 - 直接录制为 GIF 格式
4. 摄像头录制 - 录制摄像头画面或叠加到录屏上
5. 系统音频 + 麦克风录制
6. 内置修剪工具 - 录制完成后可修剪视频

### 标注/编辑功能
1. 箭头工具
2. 文字工具
3. 形状工具（矩形、圆形、线条）
4. 马赛克/模糊工具 - 隐藏敏感信息
5. 高亮标记工具
6. 画笔/自由绘制
7. 裁剪工具
8. 序号标注（自动编号的圆圈标注）

### 快捷操作
1. 快速预览悬浮窗 - 截图后在屏幕角落显示小预览，可拖拽使用
2. 钉图功能 - 将截图钉在屏幕最前方
3. 一键复制到剪贴板
4. 自动隐藏桌面图标（截图时隐藏桌面杂乱）

### 界面设计
1. 菜单栏常驻图标（相机图标）
2. 全局快捷键配置
3. 现代化 UI 设计，符合 macOS 设计规范
4. 深色/浅色模式自适应

## 目标用户
需要高效截图录屏的开发者、设计师、产品经理、内容创作者

## App 类型
Menu Bar App（菜单栏常驻 + 独立设置/编辑窗口）

## 技术要求
- Swift + SwiftUI
- macOS 13+
- Apple Silicon 支持
- 高性能，低内存占用

## 不包含的功能
- 云上传/分享（不需要）
- OCR 文字识别
- 美化背景
- 历史记录管理
