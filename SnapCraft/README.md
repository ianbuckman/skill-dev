# SnapCraft

一款功能全面的 macOS 截图录屏工具，媲美 CleanShot X，无需云服务，所有数据本地存储。

## 安装

1. 双击 `SnapCraft.dmg`
2. 将 SnapCraft 拖入 Applications 文件夹
3. 首次打开：右键 SnapCraft → 打开（绕过 Gatekeeper）
4. 授予屏幕录制权限（系统设置 → 隐私与安全性 → 屏幕录制）

## 使用方法

启动后 SnapCraft 常驻在菜单栏（相机图标），点击即可看到所有功能菜单。

### 截图功能

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 全屏截图 | ⌘⇧3 | 捕获整个屏幕 |
| 区域截图 | ⌘⇧4 | 拖拽选择区域截图 |
| 窗口截图 | ⌘⇧5 | 点击选择窗口截图 |
| 滚动截图 | ⌘⇧6 | 捕获可滚动内容 |
| 定时截图 | 菜单选择 | 3秒/5秒/10秒延迟截图 |
| 自拍截图 | 菜单选择 | 截图时叠加摄像头画面（圆形头像） |

### 录屏功能

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 全屏录制 | ⌘⇧7 | 录制整个屏幕为 MP4 |
| 区域录制 | 菜单选择 | 选择区域录制为 MP4 |
| GIF 录制 | ⌘⇧8 | 直接录制为 GIF |

- 录制时屏幕顶部会出现控制条（含计时器和停止按钮）
- 支持系统音频和麦克风同时录制
- 录制完成后可使用内置修剪工具裁剪视频

### 标注编辑

截图后点击快速预览中的「标注」按钮，进入标注编辑器：

- **箭头** — 绘制箭头指示
- **文字** — 添加文字标注（双击输入）
- **矩形/圆形/线条** — 绘制形状
- **马赛克** — 像素化区域，隐藏敏感信息
- **高亮** — 半透明高亮标记
- **画笔** — 自由绘制
- **裁剪** — 裁剪图片（带三分法参考线）
- **序号** — 自动编号的圆圈标注（1, 2, 3...）

支持撤销（⌘Z）/重做（⌘⇧Z），可选颜色和线条粗细。

### 快捷操作

- **快速预览** — 截图后屏幕右下角弹出小预览窗口，可拖拽使用
- **钉图** — 将截图钉在屏幕最前方（悬浮窗口），支持透明度调节
- **一键复制** — 截图自动复制到剪贴板
- **隐藏桌面图标** — 菜单中可切换桌面图标显示/隐藏

### 设置

通过菜单栏 → 偏好设置 访问：

- **通用** — 保存路径、图片格式（PNG/JPEG）、预览停留时间
- **截图** — 定时截图默认延迟
- **录制** — 视频帧率（24/30/60 FPS）、GIF 帧率（10/15/20 FPS）
- **快捷键** — 查看所有快捷键绑定

## 文件保存

截图和录屏默认保存到 `~/Desktop/SnapCraft/`，可在设置中更改。

文件命名格式：`screenshot_2026-03-19_14-30-25.png`

## 权限需求

| 权限 | 用途 |
|------|------|
| 屏幕录制 | 截图和录屏 |
| 摄像头 | 自拍叠加和摄像头录制 |
| 麦克风 | 录屏时录制音频 |

## 卸载

将 SnapCraft 从 Applications 拖入废纸篓。

偏好设置存储在 `~/Library/Preferences/com.local.snapcraft.plist`，可手动删除。

## 技术信息

- 技术栈: Swift 5.10 + SwiftUI
- 最低系统要求: macOS 14 (Sonoma)
- 架构: Apple Silicon (arm64)
- 二进制大小: ~1.1 MB
- 无第三方依赖，全部使用系统框架
