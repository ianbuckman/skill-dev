# SnapCraft

对标 CleanShot X 的全功能 macOS 截图录屏工具，纯本地运行，无云依赖。

## 安装

1. 双击 `dist/SnapCraft.dmg`
2. 将 `SnapCraft.app` 拖入 `Applications` 文件夹
3. 首次打开：**右键 → 打开**（未签名应用需要此步骤）
4. 首次使用需授权屏幕录制权限：**系统设置 → 隐私与安全性 → 屏幕录制 → 勾选 SnapCraft**
5. 如需录制麦克风音频，还需授权麦克风权限

## 系统要求

- macOS 14.0 (Sonoma) 或更高
- Apple Silicon (arm64)

## 功能与快捷键

### 截图

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| ⌘⇧3 | 全屏截图 | 捕获整个显示器 |
| ⌘⇧4 | 区域截图 | 拖拽选区，带十字准线和放大镜 |
| ⌘⇧5 | 窗口截图 | 点击选择窗口 |
| ⌘⇧6 | 滚动截图 | 选区后自动滚动拼接长内容 |
| ⌘⇧9 | OCR 文字识别 | 选区提取文字到剪贴板 |
| ⌘⇧0 | All-In-One | 所有模式的快速选择面板 |

**延时截图**: 从菜单栏选择，支持 3/5/10 秒倒计时
**冻结屏幕**: 冻结当前屏幕状态，精确捕获悬停菜单等动态内容

### 录屏

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| ⌘⇧7 | 屏幕录制 | 录制为 MP4 (H.264) |
| ⌘⇧8 | GIF 录制 | 直接录制为优化 GIF |

录制功能支持：
- 全屏 / 窗口 / 自定义区域
- 系统音频 + 麦克风同步录制
- 摄像头画中画叠加
- 鼠标点击高亮
- 按键显示
- 录制时自动隐藏桌面图标
- 录制时自动开启勿扰模式
- 内置视频编辑器（裁剪、音量调整）

### 标注编辑器

截图后通过 Quick Access 浮窗的「Annotate」按钮打开：

- **箭头**: 4 种样式（直线/弯曲/双向/粗线）
- **形状**: 矩形、填充矩形、椭圆、直线
- **铅笔**: 自由绘制 + 自动平滑
- **高亮**: 半透明荧光笔
- **文字**: 可自定义字体、颜色、大小
- **模糊/马赛克**: 隐藏敏感信息
- **聚光灯**: 突出强调区域
- **计数器**: 自动递增编号圆圈
- **裁剪**: 支持指定纵横比
- **撤销/重做**: ⌘Z / ⌘⇧Z

### Quick Access 浮窗

截图/录屏后自动出现的悬浮面板：
- 一键复制到剪贴板
- 直接保存
- 打开标注编辑
- 拖放到其他应用
- 可配置自动关闭时间

### Pin 钉图

将截图钉在屏幕上始终可见：
- 右键菜单调整透明度
- 方向键精确移动（Shift+方向键 10px 步进）
- 锁定模式：钉图下方应用可正常交互

### 背景美化工具

通过标注编辑器访问：
- 10 种内置渐变背景
- 可调 padding 和圆角
- Auto Balance 自动居中
- 支持导出和一键复制

### OCR 文字识别

⌘⇧9 触发，选择屏幕区域，文字自动提取并复制到剪贴板。完全本地处理，不传输数据。

## 设置

通过菜单栏 → Settings 打开：

- **通用**: 保存目录、默认格式(PNG/JPG)、命名规则、Quick Access 自动关闭时间
- **截图**: 十字准线、放大镜、光标显示、窗口阴影和 padding
- **录制**: FPS、画质、音频、鼠标点击高亮、按键显示、GIF 帧率
- **快捷键**: 查看所有快捷键映射
- **外观**: 深色/浅色/系统模式
- **预设**: 保存常用配置组合

## 数据存储

| 数据 | 位置 |
|------|------|
| 截图/录屏文件 | 用户配置目录（默认 ~/Desktop） |
| 历史记录 | ~/Library/Application Support/SnapCraft/history.json |
| 预设配置 | ~/Library/Application Support/SnapCraft/presets.json |
| 缩略图缓存 | ~/Library/Application Support/SnapCraft/thumbnails/ |
| 用户设置 | ~/Library/Preferences (UserDefaults) |

## 卸载

1. 退出 SnapCraft（菜单栏 → Quit SnapCraft）
2. 从 Applications 文件夹删除 SnapCraft.app
3. 可选：删除 ~/Library/Application Support/SnapCraft/

## 技术信息

- 版本: 1.0.0
- 构建: Swift 5.10 / SwiftUI / SPM
- 架构: arm64 (Apple Silicon)
- 大小: ~1.6 MB (DMG)
- 依赖: 零外部依赖，全部使用 Apple 原生框架
- 框架: ScreenCaptureKit, AVFoundation, Vision, CoreGraphics, ImageIO, Carbon
