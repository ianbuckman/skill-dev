# Task State — SnapCraft

## 目标（创建后不可修改）
- App 名称: SnapCraft
- 一句话描述: 一款功能全面的 macOS 截图录屏工具，媲美 CleanShot X，无需云服务，所有数据本地存储
- 核心功能:
  1. 多模式截图（区域、全屏、窗口、滚动、定时、自拍叠加）
  2. 屏幕录制（全屏/区域录制、GIF、摄像头叠加、音频、修剪）
  3. 标注编辑器（箭头、文字、形状、马赛克、高亮、画笔、裁剪、序号）
  4. 快捷操作（快速预览、钉图、复制剪贴板、隐藏桌面图标）
- App 类型: Menu Bar App
- 技术栈: Swift/SwiftUI
