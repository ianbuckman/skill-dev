# DeskPet

桌面浮动宠物，陪你工作、帮你专注。一只像素风小猫在桌面上自由走动，内置番茄钟帮助专注工作。

## 安装

1. 双击 `dist/DeskPet.dmg`
2. 将 `DeskPet.app` 拖入 `Applications` 文件夹
3. 右键点击 DeskPet → 打开（首次运行需要授权未签名应用）

## 功能

### 桌面宠物
像素风小猫在桌面上自由走动。支持多种动画状态：闲逛、坐下、睡觉、奔跑。透明窗口悬浮于所有窗口之上，跟随 Space 切换。

### 互动
- **单击**宠物：触发反应动画，心情变开心
- **拖拽**宠物：移动到任意位置
- **双击**宠物：触发旋转跳跃特效

### 番茄钟
25 分钟工作 + 5 分钟休息循环：
- 工作时宠物安静陪伴（闲坐、打盹）
- 休息时宠物活跃庆祝（奔跑、跳跃）
- 阶段切换时发送系统通知

### 心情系统
宠物心情随时间和互动变化：
- **开心**：更多奔跑和跳跃
- **普通**：正常行为
- **困倦**：更多睡觉和静坐

### 菜单栏控制
点击菜单栏猫咪图标打开控制面板：
- 启动/暂停/重置/跳过番茄钟
- 查看剩余时间和进度
- 今日专注统计（完成周期数、总分钟数）
- 显示/隐藏宠物
- 退出应用

## 数据存储

- 番茄钟统计存储在 `UserDefaults`，每日自动重置
- 无需额外配置

## 构建

```bash
swift build -c release
```

## 技术栈

- Swift 5.10+ / SwiftUI / AppKit
- macOS 14+ (Sonoma)
- Swift Package Manager

## 卸载

1. 从 Applications 中删除 DeskPet.app
2. 可选：删除 `~/Library/Preferences/com.local.deskpet.plist`
