# App Concept — DeskPet

## 目标（创建后不可修改）
- App 名称: DeskPet
- 一句话描述: 桌面浮动宠物，陪你工作、帮你专注
- 核心功能:
  1. 浮动动画角色 — 像素风小猫在桌面自由走动（闲逛/坐下/睡觉/奔跑），透明窗口悬浮于所有窗口之上
  2. 互动系统 — 点击宠物有反应动画，可拖拽到任意位置，双击触发特殊动作
  3. 番茄钟 — 25分钟工作+5分钟休息循环，工作时宠物安静陪伴，休息时宠物活跃庆祝
- 辅助功能:
  4. 心情系统 — 宠物心情随使用时间变化（开心/普通/困倦），影响动画表现
  5. Menu Bar 控制 — 从菜单栏快速启停番茄钟、显示/隐藏宠物、查看今日专注统计
- App 类型: 标准窗口 App（浮动透明窗口 + Menu Bar 辅助）
- 目标用户: 想要桌面陪伴 + 专注工具的远程工作者/学生
